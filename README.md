# The Editorial — Team Members Webpage

## 📌 Overview

**The Editorial** is a modern, responsive team showcase webpage designed to present a professional collective of strategists, writers, and designers. The layout emphasizes clarity, typography, and minimal aesthetics, making it suitable for agencies, startups, and editorial-style brands.

---

## 🎯 Features

* Responsive design for desktop, tablet, and mobile
* Clean and structured layout using CSS Grid and Flexbox
* Interactive navigation with hamburger menu (mobile)
* Team member cards with:

  * Profile image
  * Name and role
  * Short professional bio
  * Social media icons
* Hero section with strong typography focus
* Contact section for engagement
* Sticky header with smooth scrolling

---

## 🛠 Technologies Used

* **HTML5** — Semantic structure
* **CSS3** — Custom properties, Flexbox, Grid, media queries
* **JavaScript (Vanilla)** — Navigation toggle functionality
* **Google Fonts** — Inter, Poppins, Playfair Display
* **Font Awesome** — Icons

## 📱 Responsive Behavior

* **Desktop:** Full navigation with horizontal layout
* **Tablet:** Adjusted spacing and layout stacking
* **Mobile:**

  * Hamburger menu enabled
  * Navigation collapses using `max-height` animation
  * Content stacks vertically

## 🎨 Design Highlights

* Subtle hover animations and transitions
* Editorial-style typography hierarchy
* Balanced spacing and whitespace for readability

## 🔧 Key Functionality

### Navigation Toggle

* JavaScript toggles `.active` class on mobile menu
* Icon switches between menu (`bars`) and close (`times`)
